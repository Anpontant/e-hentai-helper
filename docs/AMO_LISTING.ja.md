# AMO掲載情報

## 名前

E-Hentai Helper

## 短い説明

E-Hentai / ExHentai の画像閲覧を、自動スクロール、画面フィット、先読みで快適にします。

## 詳細説明

E-Hentai Helper は、E-Hentai / ExHentai の画像ビューアをFirefoxで見やすくするための補助拡張機能です。

主な機能:

- 画像ページを開いたときに、メイン画像の上端へ自動スクロール
- 画像をブラウザの高さ・幅に合わせて表示
- 画像を原寸に近い表示へ切り替え
- 次ページを最大3ページ先まで先読み
- 先読みの状態と読み込み時間をページ右下に表示
- ポップアップから設定を変更
- ブラウザウィンドウの全画面表示を切り替え

設定はFirefox内のローカルストレージに保存されます。外部サーバーへ利用状況や個人情報を送信しません。

## 権限の説明

### storage

先読みページ数、画像表示モード、自動スクロール、状態表示の設定をFirefox内に保存するために使用します。

### activeTab / tabs

ポップアップから現在開いているE-Hentai / ExHentaiのタブへ、スクロールや設定再読み込みの指示を送るために使用します。

### https://e-hentai.org/s/_ / https://exhentai.org/s/_

画像ビューアページ上で、メイン画像の表示調整、スクロール、次ページの先読みを行うために使用します。

## データ収集

この拡張機能はユーザーデータを収集・販売・共有しません。設定はFirefox内に保存され、外部サーバーへ送信されません。
